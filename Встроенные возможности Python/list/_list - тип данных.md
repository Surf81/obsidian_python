### Необычные `фишки`

Присвоение срезу любого итерируемого объекта
```python
l = [1, 2]
m = [5, 6]
l[2:] = m
print(l)
# >>> [1, 2, 5, 6]

l[:0] = 'asf' # работает гораздо быстрее insert(0, value)
print(l)
# >>> ['a', 's', 'f', 1, 2, 5, 6]

l[7:] = set(range(50,52))
print(l)
# >>> ['a', 's', 'f', 1, 2, 5, 6, 50, 51]
```
