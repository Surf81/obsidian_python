### Необычные `фишки`

Присвоение срезу любой итерируемой последовательности
> работает гораздо быстрее insert(pos, value) - примерно в 1,5-2 раза
```python
l = [1, 2]
m = [5, 6]
l[2:] = m
print(l)
# >>> [1, 2, 5, 6]

l[:0] = 'asf'
print(l)
# >>> ['a', 's', 'f', 1, 2, 5, 6]

l[7:] = set(range(50,52))
print(l)
# >>> ['a', 's', 'f', 1, 2, 5, 6, 50, 51]

l[2:2] = ('D', 'F')
# >>> ['a', 's', 'D', 'F', 'f', 1, 2, 5, 6, 50, 51]
```
